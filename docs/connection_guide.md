# PostgreSQL + dbt Connection Guide (IT4065C)

This guide explains how to connect to the course PostgreSQL sandbox and configure dbt locally using VS Code.

You do NOT need Docker for this course.

---

## What You Will Need
- PostgreSQL connection credentials (provided by the instructor)
- Python 3.9+ installed
- VS Code installed
- Internet connection

---

## Step 1: Install dbt for PostgreSQL

Open a terminal (Command Prompt, PowerShell, or Terminal on macOS/Linux) and run:

pip install dbt-postgres
