# IT4065C Course Lab Repository

This repository contains the lab materials and templates for **IT4065C: Data Technologies Administration**.

## How to use this repo
- Go to `labs/` and open the folder for the current week (e.g., `labs/week01_preflight/README.md`).
- Follow the instructions and submit required artifacts in Canvas (or your team repo if assigned).
- dbt project files are located in `dbt/it4065c_platform/`.

## Tools used
- PostgreSQL (shared course sandbox)
- dbt (run locally in VS Code)

## Start here
- **Week 1:** `labs/week01_preflight/README.md`
- Connection help: `docs/connection_guide.md`
- Troubleshooting: `docs/troubleshooting.md`

## Academic integrity
Use templates and starter files as intended, but all submissions must reflect your own work and documentation.
See `docs/academic_integrity.md`.
